# ✊✋✌️ Jokenpô Game

Implementação do clássico jogo **Pedra, Papel e Tesoura** desenvolvida com HTML, CSS e JavaScript.

O projeto apresenta uma interface moderna e interativa, onde o jogador escolhe um personagem e enfrenta o computador em partidas com sistema de vidas e múltiplas rodadas.

---

## 🎮 Demonstração

Jogue online:

https://will10iam.github.io/jokenpo_game/

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎯 Funcionalidades

- Escolha de personagem
- Sistema de rodadas
- Sistema de vidas
- Lógica de vitória/derrota baseada nas regras do jogo
- Interface visual interativa
- Possibilidade de reiniciar partidas

---

## 🧠 Lógica aplicada

O projeto utiliza lógica de programação para:

- Gerar jogadas aleatórias do computador
- Comparar escolhas do jogador e do computador
- Controlar sistema de vidas
- Gerenciar estado das rodadas
- Atualizar interface dinamicamente
